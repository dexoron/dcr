# Первые шаги

## Создать проект

```bash
dcr new my-app
cd my-app
```

Будет создана структура:

```
my-app/
├── dcr.toml
└── src/
    └── main.c
```

`dcr.toml`:

```toml
[package]
name = "my-app"
version = "0.1.0"
type = "none"
description=""
author=""
license = "GPL-3.0-or-later"

[build]
language = "c"
standard = "c11"
compiler = "clang"
kind = "bin"

[dependencies]
```

## Build

```bash
dcr build
```

Output — `target/\<triple>/debug/my-app` (Linux) or `target/debug/my-app.exe` (Windows).

Профиль:

```bash
dcr build --release       # релизная сборка
dcr build --debug         # дебажная (по умолчанию)
```

## Запустить

```bash
dcr run
```

Собирает (если нужно) и запускает бинарник.

## First test

```bash
dcr test --init      # create test template
dcr test             # run tests
```

## Форматирование

```bash
dcr fmt              # clang-format по .clang-format
```
