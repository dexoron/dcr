# Установка

## Linux (Dexoron Packages)

Настройте менеджер пакетов на https://pkg.dexoron.su и установите:

**Arch Linux**

```sh
sudo pacman -Sy dcr
```

**Debian/Ubuntu**

```sh
sudo apt update && sudo apt install dcr
```

**Fedora/RHEL**

```sh
sudo dnf install dcr
```

## Arch Linux (AUR)

```sh
yay -S dcr
```

## macOS / Linux (Homebrew)

```sh
brew tap dexoron/dexoron
brew install dcr
# or
brew install dexoron/dexoron/dcr
```

## Snap (Linux)

```sh
sudo snap install dcrup
```

## Nix (flake)

```sh
nix run github:dexoron/dcr
# or install to profile:
nix profile install github:dexoron/dcr
```

## Cargo (crates.io)

```sh
cargo install dcr
```

## Linux / macOS (script)

```sh
curl -fsSL https://dcr.dexoron.su/install_bsd.sh | bash
```

## BSD (script)

```sh
curl -fsSL https://dcr.dexoron.su/install.sh | bash
```

## Windows (PowerShell)

```powershell
irm https://dcr.dexoron.su/install.ps1 | iex
```

## From source

```sh
git clone https://github.com/dexoron/dcr.git
cd dcr && cargo build --release
ln -sf "$PWD/target/release/dcr" ~/.local/bin/dcr
```

## Post-install

Verify the installation:

```bash
dcr setup
```

Man pages are available after installation:

```bash
man dcr
man dcr-build
```

Команда создаёт `~/.dcr/` с дефолтным `config.toml` и скачивает публичный реестр.

```toml
[[registry]]
name = "default"
url = "https://index.dcr.pm"
```
