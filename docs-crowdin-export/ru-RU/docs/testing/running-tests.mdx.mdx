# Запуск тестов

## Выполнение

```bash
Тест DCR
```

Выполняет сборку проекта, а затем компилирует и запускает тестовые файлы из каталога `tests/`.

## Профили

```bash
dcr test                     # debug (по умолчанию)
dcr test --release           # release
dcr test --debug             # явно debug
```

## Вывод

```
=====================
  Сводка результатов набора тестов
=====================
ИСПОЛЬЗОВАНО: 5
ПРОЙДЕНО: 3
ПРОПУЩЕНО: 1
НЕ ПРОЙДЕНО: 1
=====================
```

## Код возврата

* 0 — все тесты пройдены (FAIL = 0)
* 1 — есть упавшие тесты

## Что строится

* Все C/C++ файлы из `tests/`
* Include path: `tests/` (для `dcr_test.h`)
* Линкуется с основным проектом (если `type = "lib"`)
