# Системные команды

## `dcr setup`

`~/.dcr/config.toml` — конфигурация реестров

```bash
dcr setup
```

Если файл `~/.dcr/config.toml` не найден, DCR возвращает ошибку. Создайте его вручную:

```toml
[[registry]]
name = "default"
url = "https://index.dcr.pm"
```

## `dcr --help`

Отображает справку по всем командам.

```bash
dcr --help
```

## `dcr --version`

Отображает версию и целевую платформу.

```bash
dcr --version       # dcr 0.7.0 (x86_64-unknown-linux-gnu)
```

## `dcr --update`

Автоматическое обновление. Загружает последнюю версию бинарного файла из раздела «Релизы» на GitHub и заменяет текущую версию.

```bash
dcr --update
```

Возможности:

* Определяет платформу и архитектуру автоматически
* Выдает предупреждение при установке через AUR (вместо этого используйте пакетный менеджер)
* Работает на Linux, macOS, Windows
