# Команды проекта

## `dcr new \<name>`

Создает новый проект со стандартной структурой и `dcr.toml`. Аргумент `name` является обязательным.

В названии проекта допускаются только ASCII буквы, цифры и подчеркивания `_` и дефисы `-`.

```bash
dcr new my-app
```

Генерирует:

* `dcr.toml` с базовыми полями
* `src/main.c` с основным шаблоном

## `dcr init`

Инициализирует проект DCR в текущем (пустом) каталоге.

```bash
mkdir app && cd app
dcr init
```

Название проекта взято из имени текущего каталога. Имя каталога должно соответствовать тем же правилам именования, что и в `dcr new`.
